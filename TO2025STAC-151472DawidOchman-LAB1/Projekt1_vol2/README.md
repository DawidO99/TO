# Konwerter Walut NBP

Projekt realizujący konwersję walut na podstawie kursów z Narodowego Banku Polskiego.

## Struktura projektu

```
Projekt1_vol2/
├── CMakeLists.txt
├── main.cpp
├── include/          # Pliki nagłówkowe
└── src/              # Pliki implementacyjne
```
```

## Zasady SOLID

- **SRP**: Każda klasa ma jedną odpowiedzialność
- **OCP**: Interfejsy pozwalają na rozszerzanie bez modyfikacji
- **LSP**: Implementacje są zamienne
- **ISP**: Małe, wyspecjalizowane interfejsy
- **DIP**: Zależności od abstrakcji, nie konkretów
